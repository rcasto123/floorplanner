# FloorCraft by Aircall

Interactive office floor planner and seating management tool.

**Live:** [robertcas.to/projects/floorplanner](https://robertcas.to/projects/floorplanner)

## Features

- SVG-based floor plan with pan, zoom, and snap-to-grid editing
- Dark/light mode with full Aircall brand design system
- Wall drawing tool
- Drag-and-drop symbol library with 35+ furniture items
- Employee-to-seat assignment with department color-coding
- Headcount roster with search, sort, and status filters
- JSON import/export and PNG export
- Undo/redo, multi-select, alignment tools, keyboard shortcuts

## Stack

- Single-file HTML/CSS/JS — no build step
- SVG rendering for resolution independence
- localStorage for persistence
- GitHub Pages for hosting

## Deployment

Push to `main` → GitHub Actions auto-deploys to GitHub Pages.
